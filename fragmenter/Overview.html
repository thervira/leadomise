What is LEADOMISE FRAGMENTER TOOL?¶

LEADOMISE - Fragmenter Tool is part of LEADOMISE suite (leadomise.com). It is an open source and user-friendly tool wit Graphical User Interface for fast molecular fragmentation and fragment analysis.
