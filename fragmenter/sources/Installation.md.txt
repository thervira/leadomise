# Installation

The LEADOMISE Fragmenter Tool can be installed on Windows/Mac/Linux.

## Prerequisites

- python 3.6+ (http://www.python.org/) 
- matplotlib (https://matplotlib.org/) 
- rdkit (https://rdkit.org/)
- MySQL (https://www.mysql.com/downloads/) 
- MySQL python Connector (https://www.mysql.com/downloads/)
- pandas (https://pandas.pydata.org/) 


## Windows users 

### installer


### from binaries

git clone https://gitlab/leadomise/FragmenterTool.git
