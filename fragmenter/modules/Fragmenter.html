BRICS Fragmentation¶

BRICS Fragmenter.

  • Fragments the input file

  • Calculates the properties of the fragments

  • Uploads fragments (that have properties values within provided ranges) and their properties to database

class Fragmenter_brics.Fragmenter(input, username=None, password=None, minFragmentSize=1)[source]¶

Fragments the input molecule(s), calculates their properties and adds the fragments to the MySQL database called fragments and table ‘fragments’. If database ‘fragments’ and table ‘fragments’ does not exists, it creates them.

Parameters

  • input – str ligand file location to sd file

  • username – str (default: None) MySQL username

  • password – str (default: None) MySQL password

  • minFragmentSize – int (default: 1) min fragment size for BRICS fragmentation

checkMySQLAccess()[source]¶

Check MySQL connection and connect if possible. Otherwise throw an error.

fragment()[source]¶

Uses BRICS fragmentation method to cut ligands in the input file based on the minFragmentSize and stores them in the set.

Default minFragmentSize: 1

getDataFragments(minMolWeight, maxMolWeight, minHBA, maxHBA, minHBD, maxHBD, minLogP, maxLogP, minCharge, maxCharge, minHeavyAtoms, maxHeavyAtoms, minTPSA, maxTPSA, minRotBonds, maxRotBonds)[source]¶

Gets a set of properties for every fragment in the list based on the provided ranges.

List of calculated properties: fr_3d - 3d coordinates of the fragment molecule, Smiles - Smiles of the fragment molecule, hba_count - Number of Hydrogen Bond Acceptors, hbd_count - Number of Hydrogen Bond Donors, ring_count - Number of Rings present in the fragment molecule, tpsa - Topological Polar Surface Area of the fragment molecule, heavy_atoms - Number of Heavy Atoms present in the fragment, mw - Molecular Weight of the fragment, aromatic_rings - Number of Aromatic Rings, aliphatic_rings - Number of Aliphatic Rings, rot_bonds - Number of Rotatable Bonds in the fragment molecule, charge - Charge of the fragment, cLogP - calculated logP (octanol-water partition coefficient)

:returns list

list with molecular coordinates and properties

upload_fragments()[source]¶

Uses provided MYSQL username and password to connect to database ‘fragments and uploads the fragments and their properties to the database fragments and table ‘fragments’.

Throws dialog with error if provided username or password are incorrect.
