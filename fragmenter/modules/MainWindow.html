Fragmenter Tools’s GUI¶

The PyQt5 GUI for LEADOMISE - Fragmenter Tool¶

class MainWindow.MainWindow[source]¶

MainWindow comprises of four main group boxes: - Settings Group Box (for fragmentation set up), - Tab Widget (for dataframe with generated fragments for both input file fragments and database fragments), - Plot Options Group Box (with options to create plots), - Plot Box (with canvas for created plot).

Additionally, there are: - Rules Group box (for database customization) and - Similarity Search Group Box (for searching similar fragments).

Rules for customized fragmentation:

min fragment size, range of molecular weight, range of hydrogen bond acceptors, range of hydrogen bond donors, range of clogP, range of charge, range of heavy atoms, range of TPSA, range of rotatable bonds

searchSimilarFragments()[source]¶

Searches for similar fragments within both input file and database fragments based on the provided SMILES template.

Dummy atoms from the provided SMILES template are not considered.

Parameters

  • smilesTemplate – str SMILES string for similarity search

  • minTanimoto – float minimun Tanimoto similarity

  • maxTanimoto – float maximum Tanimoto similarity

Shows inputResults_model: dataframe

dataframe with similarity results withing input file fragments

Shows databseResults_model: dataframe

dataframe with similarity results withing database fragments

Result dataframes are shown in two additional tab widgets added to self.tabWidget

startFragmentation()[source]¶

Starts the Fragmenter using the input file ( and username and password for MySQL only if ‘add to database check box is checked in), as well as user-defined options for customized database.

Shows database fragments table only if the box addToDatabase is checked in when running the fragmentation. Otherwise input fragments will not be uploaded into database and database dataframe will not be shown.
